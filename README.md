# insurance-comparator
I built a web application to compare insurance quotes and help users evaluate coverage, price, and deductibles in a structured way.

Problem detected:
When people receive multiple insurance quotes, they often compare them manually using spreadsheets or notes. This process can be confusing and inefficient.
insurance-comparator simplifies this by providing a clear comparison interface where users can evaluate different insurance options side by side

Solution:
1.The application allows users to:
2.Add insurance quotes
3.Compare quotes in a structured table
4.Evaluate coverage, price, and deductible
5.Identify the best option quickly

Features:

Add and manage insurance quotes
Quote comparison dashboard
Simple scoring logic to evaluate options
Structured data storage
Clean and transparent interface
